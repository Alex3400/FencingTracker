# Haverstock Fencing ELO Ratings Website

This is the static website that displays ELO ratings for Haverstock Fencing Club.

## Setup

This site is designed to be hosted on **GitHub Pages**.

### To enable GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (e.g., `main`)
4. Select `/docs` as the folder
5. Click "Save"

Your site will be live at: `https://alex3400.github.io/FencingTracker/`

## Updating the Data

To update the website with new tournament data:

1. Run the Python script to process new data:
   ```bash
   cd src
   python3 track_match_history.py
   ```

2. This will generate fresh JSON files in `docs/data/`

3. Commit and push the changes:
   ```bash
   git add docs/data/
   git commit -m "Update ratings after [date] tournament"
   git push
   ```

4. GitHub Pages will automatically rebuild the site (takes 1-2 minutes)
