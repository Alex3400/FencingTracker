About the ELO System

An ELO rating system assigns each fencer a numerical score representing their skill level. When two fencers compete, the system predicts who should win based on their current ratings and updates both ratings after the match based on the actual result.

Example: If Alice (rated 1600) beats Bob (rated 1400), Alice gains only a few points because she was expected to win. But if Bob beats Alice, he gains many more points because he exceeded expectations.

Over time, these adjustments cause ratings to stabilize and accurately reflect each fencer's true skill level.
